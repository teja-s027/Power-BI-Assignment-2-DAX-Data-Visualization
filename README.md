# Power BI Assignment 2 – DAX & Data Visualization

## Project Overview
This project analyzes E-Commerce Sales Data using Power BI and DAX.

## Files
- Power BI Assignment 2.pbix
- PowerBI_Assignment2_Report.docx

## DAX Calculations
- Category Type
- Revenue per Order
- Sales Category
- Order Count
- Average Profit in Delhi
- YTD Sales

## Visualizations
- Sales Target Achievement by Category
- Max Profit Margin by Sub-Category
- Monthly Sales Trend
- Profit and Quantity by Sub-Category
- Sales Performance Matrix
- Geographic Sales Analysis
- Sales Distribution by Sub-Category
- Order Count Analysis by State
